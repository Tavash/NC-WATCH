Have I Been Pwned

Cet outil permet de vérifier si une adresse mail est présente dans la base de données d’un site qui a été piraté un jour.

Pour accéder à cette information il suffit de rentrer dans le champ "Email" l'adresse que l'on souhaite vérifier puis de cliquer sur le bonton a sa droite.

Sur l'exemple ci-dessous on peut malheureusement que l'adresse à été utilisé par des sites piratés, la liste des ces sites s'affiche en dessous,puis une liste des données compromises est affichée pour chaque site (“compromised data”).

Built With

Built With permet de recenser les informations sur les technologies utilisées sur le site internet de l’entreprise. On retrouve ainsi des informations sur le serveur, l’hébergeur, le fournisseur du certificat SSL, les langages utilisés, les plugins etc...

Pour utiliser cet outil il suffit de saisir un nom de domaine valide et cliquer sur “GO BUILT WITH”.

Le résultat de la recherche est présentée sous forme de liste avec tous les aspects technologiques

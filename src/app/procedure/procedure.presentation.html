Présentation

Network Company Wtach est un site web qui a été créé dans le cadre d'un projet de Master 2 MIAGE et qui a pour objectif de récupérer toutes les informations publiques présentes sur Internet (OSINT : Open-Source INTelligence gathering)

Ce dernier est constitué de plusieurs API Open Source qui ont été intégrée au site (onglet "tools"). Ces derniers sont détaillés dans la partie "Outils" de la procédure et sont les suivantes :

  • CRAWLER
  • HAVE I BEEN PWNED
  • WHO IS
  • SHODAN
  • GOOGLE DORKS
  • SSL
  • BUILT WITH

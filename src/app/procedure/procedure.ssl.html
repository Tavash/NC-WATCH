SSL

Cette API permet de récupérer les informations sur la configuration du SSL à partir d’un nom de domaine

Il suffit d’entrer un nom de domaine valide puis de cliquer sur le bouton "CHECK" pour lancer une recherche. Le premier lancement de la recherche s’avère très long et ne donne pas de résultat à cause du “Time Out”. Cependant, une à deux minutes après avoir lancé la première recherche qui ne fournit aucun résultat, un second clic sur “CHECK” permet d’afficher les résultats qui sont stockés en cache.
